This script samples an image from a conditional 4-class 128x128 DCGAN (deep convolutional generative adversarial network trained on a dataset of human, dog, cat, and wilflife faces.

Inference on the model uses Tensorflow.js, entirely client-side. The model itself is ~7.8 MB and is downloaded by the client from this webserver, but could be relocated to IPFS.

The model takes in a random 100-dimensional vector (generated using nilRandom()) and a single class label (e.g. 'person', 'dog', 'cat', 'wild') and deterministically outputs the face at that point in the model's latent space.

If the code and model are hosted in a decentralized way (e.g. on-chain and/or on IPFS), immutability and reproducibility are ensured, as the model will always generate the same face given the same input vector. In a sense, you own a point in the latent space.
